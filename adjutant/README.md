# Adjutant

A compact pixel-art Adjutant companion for Codex, inspired by StarCraft: Brood War. Adjutants are a form of non-sapient artificial intelligence utilized by the terrans.

## Install

Copy this folder into your Codex pets directory:

```text
C:\Users\<you>\.codex\pets\Adjutant
```

The folder should contain:

```text
pet.json
spritesheet.webp
```

Then restart Codex or reload the pet picker if needed.

## Files

- `pet.json` - Codex pet manifest.
- `spritesheet.webp` - 9-row Codex pet animation atlas.
